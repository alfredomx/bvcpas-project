# v0.13.0 — Manage portals (CRUD del catálogo)

**Estado**: ✅ Cerrado: 2026-06-13
**App version**: `0.12.0 → 0.13.0`
**Módulo principal**: 19-bank-accounts
**Backend**: bank-worker mapi (`/v1/banking/portals`)

## Motivación

El catálogo de portales (`/v1/banking/portals`) solo tenía el wrapper de
**listar** (`listBankPortals`); no había UI para administrarlo. El
dropdown "Portal" del form de login salía vacío cuando la tabla no tenía
datos, y no había forma de darlos de alta desde el front.

Además, "portal" no es solo "banco": hay portales de utilities, payroll,
etc. Por eso la UI se llama **"Manage portals"** (no "Add bank").

## Cambios

### API (`bank-accounts.api.ts`)

- `createBankPortal` (POST), `updateBankPortal` (PATCH
  `/v1/banking/portals/:portalId`), `deleteBankPortal` (DELETE).
- Tipos `BankPortalDetail`, `CreateBankPortalBody`, `UpdateBankPortalBody`.

### Hooks

- `useCreateBankPortal`, `useUpdateBankPortal`, `useDeleteBankPortal`
  (TanStack mutations; invalidan `['bank-portals']` + toast).

### UI

- Botón **"Manage portals"** en el header de `/bank-accounts` (siempre
  visible — el catálogo es global, no por cliente).
- `<ManagePortalsSheet>`: sheet lateral con la lista de portales,
  **buscador por nombre** (hay ~280), botón "Add", y editar / borrar por
  fila. Cada portal muestra nombre + URL (link externo).
- `<BankPortalFormDialog>`: crea **y** edita (name requerido, portalUrl
  opcional validada http/https).
- `<DeletePortalDialog>`: confirmación con `<AlertDialog>`. El backend
  rechaza borrar un portal en uso (FK); el toast de error lo comunica.

## Decisiones de esta versión

- **D1** — Nombre "Manage portals" en vez de "Add bank": un portal es
  cualquier proveedor de login (banco, utility, payroll…), no solo banco.
- **D2** — CRUD dentro de un sheet con buscador (no una tabla en la
  pantalla principal): el catálogo es grande (~280) y secundario al flujo
  principal (credenciales por cliente).
- **D3** — Solo se valida URL en el front (http/https); unicidad de
  nombre y FK en delete las maneja mapi.
